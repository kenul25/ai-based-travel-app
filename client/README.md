# Travel App Client

Expo Router mobile client for the travel app.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, choose the target you want to run:

- development build
- Android emulator
- iOS simulator
- Expo Go

The main route files live in `app/`, shared UI in `components/`, app state in `context/`, and API access in `services/`.
